# 6obcy-bridge

Connecting two [6obcy](https://6obcy.pl/) users together and tracking their conversation. So funny.

## How to run

1. Clone repository

```
git clone https://github.com/MiXerek/6obcy-bridge && cd 6obcy-bridge
```

2. Install required packages

```
npm install
```

3. Run!

```
node src/index.js
```
